# 🍕 Pizzaria di Oliveira – Página Web com Bootstrap

Este projeto foi desenvolvido como parte de uma **atividade prática da disciplina de Desenvolvimento Full Stack**.
O objetivo da atividade foi aplicar conceitos básicos de **estruturação de páginas web, utilização de frameworks front-end e organização de layouts responsivos**.

## 📚 Objetivo da atividade

A prática teve como foco:

* Estruturar um documento HTML corretamente
* Integrar o framework **Bootstrap**
* Utilizar o **sistema de grid** para organizar o conteúdo
* Aplicar **componentes de interface do Bootstrap**
* Desenvolver uma página **responsiva**
* Melhorar a experiência do usuário com elementos visuais e navegação

## 🧩 Tecnologias utilizadas

* HTML
* CSS
* Bootstrap 

## 🏗 Estrutura da página

A página simula o site de uma pizzaria fictícia chamada **Pizzaria di Oliveira** e contém as seguintes seções:

### Navbar

Menu de navegação responsivo com links para as principais seções da página.

### Cabeçalho (Header)

Área de destaque da página com:

* Nome da pizzaria
* Descrição
* Botões de ação

### Seção de destaque

Breve apresentação da pizzaria e seus diferenciais.

### Pizzas mais pedidas

Seção organizada com o **grid do Bootstrap**, contendo:

* Imagem das pizzas
* Nome
* Descrição
* Preço
* Botão para realizar pedido

As pizzas exibidas são:

* Pizza de Calabresa
* Pizza de Frango com Catupiry
* Pizza Marguerita

### Formulário de pedido

Formulário simples onde o usuário pode:

* Informar nome
* Informar telefone
* Escolher uma pizza

### Rodapé

Contém informações básicas da pizzaria.

## 📱 Responsividade

A página foi construída utilizando o **sistema de grid do Bootstrap**, garantindo que o layout se adapte a diferentes tamanhos de tela, incluindo:

* computadores
* tablets
* dispositivos móveis
